#HR Analytics Dashboard & Attrition Analysis
An interactive Power BI dashboard that provides actionable insights into employee performance, job satisfaction, workforce demographics, and critical attrition drivers across various departments, job roles, and tenure groups.

üß† Project Overview
This project focuses on building a powerful HR Analytics Dashboard using Power BI for visualization and Python (Jupyter Notebook) for data cleaning, transformation, and analysis. The dashboard is designed to give HR professionals and business stakeholders a clear, data-driven overview of the workforce's health and key performance indicators (KPIs) through interactive visualizations.

Key areas covered include:

Employee Attrition Analysis: Identifying high-risk groups by department, job role, and over time.

Performance and Job Satisfaction: Analyzing the relationship between performance ratings, job satisfaction, and employee tenure.

Workforce Demographics: Distribution analysis by gender, marital status, and job level.

Compensation and Tenure Insights: Correlating tenure, job level, and monthly income trends.

The dashboard provides both high-level summaries and in-depth analysis, enabling strategic decisions to improve employee experience, retention, and overall workforce productivity.
